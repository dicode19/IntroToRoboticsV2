Встановлення інструментів програмування
================================

У цьому курсі використовуються дві мови програмування: Blockly та Python.
Обидві мови програмуються на роботі XRP за допомогою XRPCode, інтерактивного онлайн-інструменту
розробки, який можна використовувати для розробки програм на будь-якій з цих мов.

Більш детальну інформацію про використання XRPCode дивіться в розділі «Посібник користувача XRP» на
XRPCode <https://xrpusersguide.readthedocs.io/en/latest/course/XRPCode.html>`_. 

Blockly
-------

Це графічна система програмування типу «перетягни та кинь», схожа на Scratch. Blockly — це
хороший вибір для програмування робота, особливо для учнів, які мають невеликий або взагалі не мають
досвіду програмування.  



Python
------

Більш широко використовувана мова, яка добре масштабується для великих програм і працює з професійними
інструментами програмування, такими як Visual Studio Code та багато інших. Інструментом, який буде використовуватися
протягом цього курсу для програмування на Python, є XRPCode, хоча для написання коду CircuitPython, який
використовується роботом XPR, можна також використовувати більш професійний
інструмент, такий як Visual Studio Code.

.. Примітка::

   Однією зі стратегій, яка допоможе учням швидше освоїти програмування, є
   початок з
   Blockly, щоб опанувати основні поняття, такі як функції, умовні оператори, цикли
   та базове програмування роботів. Після того, як учні ознайомляться з цими поняттями,
   перехід до Python для завершення курсу буде легким,
   особливо для деяких більш складних завдань, таких як фінальний проект.
   Це дозволить швидко освоїти матеріал, не вивчаючи занадто багато нових
   понять одночасно. Але рішення буде залежати від рівня та досвіду
   студентів, які проходять курс.
